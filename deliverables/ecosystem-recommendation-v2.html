02 - The Pitch

The Problem

App users open 1.5x per month. 66% buy once and never return. Zero off-platform presence. When users leave the app, Groupon disappears.

The Solution

A Safari extension that shows codes + cashback on every shopping site. Groupon Bucks earned off-platform can only be redeemed in the app - pulling users back.

The Impact

$4-16M/yr from 8-10M app users we already own. No acquisition needed. Every shopping session becomes a Groupon touchpoint.

04 - The Flywheel

The Cross-Pollination Flywheel

Affiliate coupons and core deals reinforce each other when both live in the app.

User needs Nike code → Opens Groupon app → Sees local deal → Books a spa → Earns Bucks → Comes back for next code

Every coupon lookup is an exposure to a core deal. Every core deal visit is an exposure to coupons. The wallet connects both.

05 - Groupon Bucks

Groupon Bucks - The Savings Wallet

A visible, growing balance that accumulates from coupon use, cashback, referrals, and streaks.

Users check their balance even when not shopping. Credits earned from affiliate coupons can be spent on core deals. This is what makes the flywheel spin.

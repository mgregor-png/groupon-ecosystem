# Mobile Team Data Request — Safari Extension Sizing

**Purpose:** Validate app usage baseline and size Safari extension opportunity for MBR
**Status:** Draft — not yet sent

## App Usage (US, last 6 months)
- DAU / WAU / MAU
- Sessions per user per month (median, P25, P75)
- DAU/MAU ratio (stickiness) trend
- Median days between sessions

## Retention
- D1 / D7 / D30 / D90 retention by recent install cohort
- Monthly retention curve out to 12 months
- Reactivation rate (% of 90d+ lapsed users who return)

## Install Base
- Total US app installs (cumulative)
- Current monthly active US users
- iOS vs Android split
- % of iOS users on iOS 15+ (Safari Web Extension minimum)

## Transaction Behaviour
- Purchases per user per year (distribution)
- ARPU (revenue per user per year)
- % of users who purchase in any given month
- Average order value

## Push Notifications (US iOS)
- Push opt-in rate
- Push CTR by notification type
- Push → purchase conversion rate

Coupons → Groupon Local

Promo banner concepts to convert anonymous Coupons visitors into registered Groupon users. Deal-led, vertical-specific — not generic discounts.
Martin Gregor · March 2026 · Cashback / Ecosystem Initiative · Mock-ups for internal review
Why vertical-specific beats generic
The promo team proposed "25% off Local, max $50, no Purple Prices." That's a start — but Coupons users are deal-savvy, high-intent visitors. A vague discount on "Local" means nothing to someone browsing Nike coupons. Show them what they can get.
Current proposal

"25% off Local deals"

Too vague. "Local" isn't a category users identify with. No visual hook, no specific deal to anchor desire. Requires the user to imagine what they'd use it for. Low click-through expected.

Proposed approach

"60% off Deep Tissue Massage near you"

Specific deal, specific saving, immediate desire. Contextual to what they're browsing (beauty coupons → spa deals). The discount is baked into the deal — no promo code needed. Registration is the gate.

Key insight: We don't need a promo code at all for v1. Groupon Local deals already have 40-70% markdowns built in. The "promo" IS the deal itself — the user just needs to register to claim it. This sidesteps the entire promo complexity (code length limits, Purple Prices feasibility, max discount caps).

The key argument: skip the promo code

The promo team's proposal (25% off, max $50, codename <10 chars, Purple Prices debate) adds complexity that isn't needed for v1.

Groupon Local deals already carry 40-70% discounts. The "promo" is the deal itself. We don't need to layer an additional promo code on top — we just need to surface the right deals to the right users at the right moment.

The registration gate IS the conversion mechanism. User sees a compelling deal → clicks → hits registration wall → signs up → becomes a Groupon user. Simple.

If we want to add a sweetener later (e.g., "$5 off your first Groupon deal"), that can be Phase 2 once we have baseline registration conversion data from Phase 1.

Phase 1: Deal-led banners → registration wall → user acquired.
Phase 2: Add promo incentive if registration rate needs boosting.

Making "25% off Local" feel specific
The promo is set: 25% off Local, max $50, code GPNLOVE. The question is how to present it so Coupons users actually care. The banners below all anchor "25% off" to concrete examples — showing what $39 massage becomes with the code, what a $60 dinner becomes. The code is the hook, but the deal example is the desire.

Design approach: Every banner below uses an existing Figma component from GC Page Design. No new components needed. All static — no dynamic deal feeds. Just swap copy and images. The promo team can have the code live immediately; engineering just places the banners.
